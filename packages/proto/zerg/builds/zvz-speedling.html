ZvZ Speedling Rush

Matchup: Zerg vs Zerg

Difficulty: Beginner

This aggressive early game build focuses on overwhelming your opponent with fast zerglings before they can establish proper defenses. Essential for ZvZ matchups.

Back to Zerg Builds

Build Order

  1. 9 Overlord
  2. 12 Spawning Pool
  3. 11 Extractor
  4. 13 Overlord
  5. @Pool: 6 Zerglings, Queen
  6. @Pool: Metabolic Boost (Speed)
  7. 18 Hatchery (if defending)
  8. 19 Overlord
  9. Continue Zergling production
  10. Pull drones off gas after speed starts

Execution Guide

Opening (0-3 minutes)

  • Scout at 9 supply with your Overlord
  • Look for opponent's pool timing
  • Drone to 11, then build Extractor
  • Don't over-drone - need larvae for units

Mid Game (3-5 minutes)

  • Start speed upgrade immediately when pool finishes
  • Produce 6 zerglings right away
  • Pull 2 drones off gas after speed starts
  • Send zerglings to opponent's base

Attack Phase (4-6 minutes)

  • Attack when speed finishes
  • Target drones first, then buildings
  • Use speed to surround enemy units
  • Continue reinforcing with more zerglings

Key Strategic Points

  • Scout Information: Check opponent's pool timing and drone count
  • Gas Management: Only mine gas for speed, then pull off
  • Larvae Management: Save larvae for zerglings, not drones
  • Attack Timing: Hit as soon as speed finishes
  • Micro: Surround enemy units and workers

Scouting Information

What to Look For

  • Pool First: Standard defensive build
  • Hatch First: Economic build - exploit with aggression
  • Gas Timing: Early gas = speed coming
  • Drone Count: Too many drones = vulnerable

Responses to Scout

  • If opponent pools first: Match their timing
  • If opponent goes hatch first: All-in with speed
  • If opponent has early gas: Expect speed wars

Micro Techniques

  • Surround: Use speed to get around enemy units
  • Focus Fire: Target single units to reduce DPS
  • Retreat: Pull damaged units back
  • Reinforcement: Rally new units to the fight
  • Worker Harassment: Always prioritize drones

Common Mistakes

  • Building too many drones early
  • Not scouting opponent's build
  • Attacking before speed finishes
  • Poor micro during engagements
  • Not following up with expansion if rush fails

Defending Against Speedling Rush

  • Match opponent's pool timing
  • Build spine crawlers if needed
  • Use queens for defense
  • Don't over-drone vs early pool
  • Wall off with buildings if possible

Transition Options

If Rush Succeeds

  • Continue pressure with more zerglings
  • Expand behind the attack
  • Tech to Banelings for follow-up

If Rush Fails

  • Retreat and build hatchery
  • Focus on economy
  • Prepare for opponent's counter-attack
  • Tech to Roaches or Banelings

Advanced Tips

  • Use Overlords to spot opponent's army movement
  • Build spine crawlers at home if over-committing
  • Consider Baneling nest for follow-up pressure
  • Practice zergling micro vs AI
  • Learn to read opponent's defensive setup
