Dark Templar Rush

Matchup: All matchups

Difficulty: Intermediate

A cheese strategy that can win games quickly if the opponent lacks detection. This high-risk, high-reward build focuses on getting cloaked Dark Templars to the enemy base before they can prepare adequate defenses.

Execution Guide

Opening (0-4 minutes)

  • Execute standard gateway opening
  • Take gas early for Dark Shrine
  • Scout opponent's tech choices
  • Hide your tech path from enemy scouts

Tech Phase (4-6 minutes)

  • Build Dark Shrine in hidden location
  • Chronoboost Dark Shrine if needed
  • Add second Gateway
  • Continue Probe production
  • Scout for enemy detection preparations

Attack Phase (6+ minutes)

  • Attack with first 2 Dark Templars
  • Target workers and key buildings
  • Continuously warp in more DTs
  • Retreat if enemy gets detection

Key Strategic Points

  • Stealth is Critical: Hide Dark Shrine from scouts
  • Detection Check: Constantly scout for enemy detection
  • Target Priority: Workers > Production > Tech buildings
  • Timing: Hit before detection is ready
  • All-in Commitment: Must deal significant damage

Dark Templar Usage

Target Priority

  1. Workers: Primary economic damage
  2. Production: Barracks, Gateways, Hatcheries
  3. Tech Buildings: Delay enemy upgrades
  4. Supply: Pylons, Supply Depots, Overlords
  5. Key Units: Queens, Medivacs, important units

Micro Techniques

  • Hit and Run: Attack and reposition frequently
  • Spreading: Don't clump DTs together
  • Worker Focus: Always prioritize worker kills
  • Retreat Timing: Leave immediately when spotted
  • Multi-tasking: Attack multiple locations

Matchup Specific Strategy

vs Terran

  • Targets: SCVs, production buildings
  • Watch For: Engineering Bays, Missile Turrets
  • Timing: Before Turrets are built
  • Escape: Avoid Marines and Scans
  • Follow-up: Gateway units or tech transition

vs Protoss

  • Targets: Probes, Pylons, tech buildings
  • Watch For: Robotics Facility, Observers
  • Timing: Before Observer completes
  • Escape: Avoid Photon Cannons
  • Follow-up: All-in or defensive play

vs Zerg

  • Targets: Drones, Queens, Overlords
  • Watch For: Spore Crawlers, Overseers
  • Timing: Before Lair tech or Spores
  • Escape: Avoid Queen anti-air
  • Follow-up: Continue pressure or expand

Scouting and Information

What to Look For

  • Detection Buildings: Engineering Bay, Robotics Facility
  • Early Defense: Cannons, Turrets, Spores
  • Economic Builds: Fast expansion, heavy worker production
  • Tech Choices: What opponent is prioritizing

Ideal Targets

  • Greedy economic builds
  • Fast expansion attempts
  • Players with no early detection
  • Tech-heavy strategies

Hiding Your Strategy

  • Dark Shrine Placement: Build in main base corner
  • Normal Play: Continue standard unit production
  • Fake Tech: Start other tech buildings as decoys
  • Scout Timing: Don't let enemy see Dark Shrine
  • Gateway Units: Build some normal units first

Common Mistakes

  • Dark Shrine spotted by enemy scout
  • Attacking into prepared detection
  • Clumping Dark Templars together
  • Not targeting workers efficiently
  • Over-committing against heavy defense
  • No backup plan if rush fails
  • Poor retreat timing

Advanced Techniques

  • Proxy Dark Shrine: Build closer to enemy base
  • Multiple Angles: Attack from different directions
  • Warp Prism Support: Mobile reinforcement
  • False Retreat: Pretend to leave, then return
  • Hidden Reserves: Keep some DTs for defense

Dealing with Detection

Mobile Detection

  • Observers: Focus fire first
  • Overseers: Target with other units
  • Scans: Retreat when area is revealed
  • Ravens: Avoid detection radius

Static Detection

  • Turrets/Cannons: Avoid or destroy
  • Spore Crawlers: Attack from different angles
  • Multiple Bases: Hit undefended areas

Transition Options

If Rush Succeeds

  • Continue DT pressure
  • Add more Dark Shrines
  • Expand behind the attack
  • Tech to Archons or other units

If Rush Fails

  • Transition to Archons (merge DTs)
  • Add Robotics or Stargate tech
  • Focus on economy and defense
  • Use remaining DTs for harassment only

Economic Considerations

  • Gas Investment: Dark Shrine costs 150/150
  • DT Cost: Each DT costs 125/125
  • Payoff Requirement: Must kill 6+ workers to break even
  • Risk Factor: Can fall behind if unsuccessful

Defending Against DT Rush

  • Early Scouting: Check for Dark Shrine
  • Detection: Build Observers/Turrets/Spores early
  • Unit Positioning: Keep units spread out
  • Counter-Attack: Hit Protoss while committed
  • Static Defense: Cannons/Turrets at key locations

Map Considerations

Good Maps for DT Rush

  • Maps with multiple attack paths
  • Large maps with hiding spots
  • Maps with exposed natural expansions
  • Complex terrain for hiding Dark Shrine

Difficult Maps

  • Small rush distance maps
  • Open maps with clear sightlines
  • Maps with easy defensive positions
  • Limited approach routes

Practice Drills

  • Practice hiding Dark Shrine placement
  • Work on DT micro vs detection
  • Practice targeting worker lines efficiently
  • Learn optimal retreat timings
  • Practice transitioning after failed rush
