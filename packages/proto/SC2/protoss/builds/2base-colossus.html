2-Base Colossus

Matchup: PvT, PvZ

Difficulty: Intermediate

A powerful mid-game timing attack that uses Colossi to break through enemy lines with superior range and splash damage. This build focuses on getting 2-3 Colossi with supporting gateway units for a devastating push.

Execution Guide

Early Game (0-6 minutes)

  • Focus on clean build order execution
  • Chronoboost Warpgate research
  • Use first Zealot for defense and scouting
  • Get Observer out first for vision
  • Continue Probe production

Tech Phase (6-10 minutes)

  • Build Robotics Bay immediately after Observer
  • Add 2 more Gateways for unit production
  • Chronoboost first Colossus heavily
  • Research Extended Thermal Lance when possible
  • Scout enemy composition with Observer

Attack Phase (10-13 minutes)

  • Attack when you have 2-3 Colossi
  • Bring supporting gateway units (Stalkers/Zealots)
  • Keep Colossi protected behind other units
  • Focus fire on clumped enemy units
  • Continuously warp in reinforcements

Unit Composition

Core Army

  • Colossi: 2-3 for the timing attack
  • Stalkers: 60-70% of gateway units
  • Zealots: 30-40% of gateway units
  • Sentries: 1-2 for Force Fields

Support Units

  • Observer: For vision and detection
  • Warp Prism: For mobility (optional)

Key Strategic Points

  • Timing: Hit before enemy gets hard counters
  • Protection: Keep Colossi safe behind meat shield
  • Range: Use Extended Thermal Lance advantage
  • Reinforcement: Continuously warp in units
  • Positioning: High ground advantage when possible

Colossus Micro

Positioning

  • Back Line: Keep Colossi behind other units
  • High Ground: Use terrain advantages
  • Spread: Don't clump Colossi together
  • Retreat Path: Always have escape route
  • Range Advantage: Attack from maximum range

Target Priority

  • Clumped Units: Marines, Zerglings, Zealots
  • Light Units: Focus on high-count armies
  • Workers: If opportunity presents itself
  • Key Units: Sentries, Medivacs, Queens

Matchup Specific Strategy

vs Terran

  • Target: Marines, Marauders, SCVs
  • Watch For: Vikings, Medivac drops
  • Timing: Before Viking count gets too high
  • Support: Add Phoenix if facing many Vikings
  • Positioning: Use range to kite bio armies

vs Zerg

  • Target: Zerglings, Hydralisks, Roaches
  • Watch For: Corruptors, Mutalisks
  • Timing: Before Spire tech or Hive units
  • Support: More Stalkers vs Corruptors
  • Positioning: Control key choke points

Common Mistakes

  • Getting Colossi caught by Vikings/Corruptors
  • Not protecting Colossi with gateway units
  • Attacking too early with only 1 Colossus
  • Poor positioning during engagements
  • Not getting Extended Thermal Lance upgrade
  • Forgetting to continue unit production during attack
  • Not scouting for enemy counters

Scouting Information

What to Look For

  • Anti-Air Preparation: Starports, Spires
  • Unit Composition: What army is enemy building
  • Tech Choices: Fast air units or heavy ground
  • Expansion Timing: Economic vs military focus

Responses to Scout

  • If heavy anti-air: Add Phoenix or delay attack
  • If pure ground: Continue with standard timing
  • If fast expansion: Push earlier with 2 Colossi
  • If air harassment: Add Stalkers for defense

Upgrade Priority

  1. Warpgate: Essential for unit production
  2. Extended Thermal Lance: Increases Colossus range
  3. +1 Attack: Increases overall army damage
  4. Charge: If using many Zealots
  5. Blink: Stalker mobility upgrade

Advanced Techniques

  • Colossus Drops: Use Warp Prism for positioning
  • Force Field Splits: Divide enemy army
  • High Ground Control: Use terrain advantages
  • Feint Attacks: Fake retreats to bait enemy
  • Multi-pronged Pressure: Attack multiple locations

Defending Against Counters

vs Vikings/Corruptors

  • Add Phoenix or Void Rays
  • Keep Colossi protected
  • Focus fire air units first
  • Use Stalkers for anti-air support

vs Immortals

  • Use gateway units to focus Immortals
  • Avoid direct Colossus vs Immortal fights
  • Surround with Zealots
  • Use Force Fields to isolate

Transition Options

If Attack Succeeds

  • Continue pressure with more Colossi
  • Take third base behind attack
  • Tech to High Templar or Carriers

If Attack Fails

  • Defend and tech up
  • Add air units for support
  • Transition to different tech path
  • Focus on economy and upgrades

Practice Drills

  • Practice clean build order execution
  • Work on Colossus positioning vs different armies
  • Practice protecting Colossi from air units
  • Learn optimal attack timings
  • Practice Force Field usage with Sentries
